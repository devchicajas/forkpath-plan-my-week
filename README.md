# ForkPath 🌿

Weekly meal plans that fit your schedule, budget, and pantry.

Turn "what's for dinner this week?" into a 2-minute decision.

**Live site:** https://forkpath-plan-my-week.lovable.app

## About

ForkPath is a weekly meal planning app for busy people. Tell it about your
week, your dietary needs, and what's already in your fridge — it generates
seven days of meals plus a categorized grocery list and a prep schedule.

## Features

- **Dietary preferences** — vegetarian, gluten-free, allergies, picky eaters
- **Pantry-aware** — uses what you already have first
- **Family mode** *(coming soon)* — different preferences per eater
- **Grocery list** — categorized by store section, shareable in a tap

## Roadmap

- **Family mode** — shared meal plans and grocery lists for households (Q1 2026)
- **Vacation pause** — pause billing when you're away from the kitchen
- **Hard allergy filters** — strict exclusion of allergens from all suggestions

## Tech stack

- TanStack Start (React 19, Vite 7)
- Tailwind CSS v4
- Built with [Lovable](https://lovable.dev)

## Development

```bash
bun install
bun dev
```

The app runs at `http://localhost:5173`.

## Editing

You can edit this project two ways:

1. **In Lovable** — changes auto-sync to this repo.
2. **Locally / on GitHub** — pushes flow back into Lovable automatically.
